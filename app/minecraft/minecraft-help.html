Installing Minecraft Forge

  1. Follow the steps above to install Minecraft classic, then choose a desired mods server to play on. Download the appropriate version of the Forge Installer.
    • Forge 9.10.0.804 for Minecraft 1.6.2 - for servers on port 25566 / 25567 / 25570
    • Forge 9.11.1.965 for Minecraft 1.6.4 - for server on port 25568
    • Forge 9.11.1.942 for Minecraft 1.6.4 - for server on port 25569
  2. Assuming Java is installed, open the X.X.X-forge-installer with the Java Runtime to open the installer. Choose "Install client" and complete the installation.
    • If Java is not installed, how did you expect to play Minecraft? Go download Java, already!
  3. In the Minecraft launcher, choose the "New Profile" button near the bottom left.
  4. Name the new profile something like Forge X.X.X.
  5. Specify Use version: release X.X.X-ForgeY.Y.Y.Y where X.X.X is the Minecraft version and Y.Y.Y.Y is the Forge version.
    • NOTE: If you need to allocate more memory to Minecraft to achieve better performance, you may need to specify it in the JVM Arguments.
      For example: to allocate a specific maximum amount of MB
      JVM Arguments: -Xmx4096M
      or to allocate a specific maximum amount of GB
      JVM Arguments: -Xmx4G
  6. Save the new profile and press Play. This should install any files necessary to run mods and open Minecraft and allow you to play the game.
  7. Only AFTER the Forge install completes, install the appropriate mod pack. See below for a list of mod packs
  8. Click "Multiplayer" and add one of the Forge servers below to begin playing with mods. Name the server whatever you want, and set its address to one listed below.

Installing a ModPack

  1. Download a ModPack archive below for the server on which you wish to play.
  2. The archives are named mods-255XX.zip where 255XX is the port number of the desired server.
  3. Install a ModPack by unzipping or extracting the archive into the folder at: %appdata%\Roaming\.minecraft\mods
  4. Once the mods are in the mods folder, you should be ready to Play Minecraft!
    • NOTE: If the game fails to start after this point, try deleting the contents of your mods folder and adding the mods back one by one.
      If all else fails, save a copy of your .minecraft folder, delete the entire folder, and restart the Minecraft installation steps from the beginning. This will clean out your Minecraft versions and mods, allowing you to re-download them from the patcher.
REMINDER: It is always a good idea to make a backup of your mods folder before changing a stable configuration.
